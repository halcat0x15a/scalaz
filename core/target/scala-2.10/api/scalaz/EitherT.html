EitherT

sealed trait EitherT[F[+_], +A, +B] extends AnyRef

Represents a computation of type F[A \/ B].

Example:

val x: Option[String \/ Int] = Some(\/-(1))
EitherT(x).map(1+).run // Some(\/-(2))
